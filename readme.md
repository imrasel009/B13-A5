1️⃣ What is the difference between var, let, and const?
Ans : Var can be redeclared, the value of let can be changed, the value of const can not be changed.

2️⃣ What is the spread operator (...)?
Ans : Spread operator is used to open the elements of an array or object.

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans : ForEach goes through every element but does not change anything, map goes through every element and make a copy and can make changes, filter goes through every elements and gets only the selected elements.

4️⃣ What is an arrow function?
Ans : Arrow function is a minimal way of writing a function.

5️⃣ What are template literals?
Ans : It is mainly backticks and inside this we can directly insert variables.
